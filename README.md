# local-ai-search
A project to manage medias (images &amp; videos) locally using AI


- The application needs to be local. 
- It could be A desktop application? or a console application. I would prefer desktop. 
- It would be great if a work On Linux, Mac. And Windows. 
- The application will be using AI To Tag and describe the images or the video in folder and sub-folder
- While Developing will be using Reka API Hosted: https://api.reka.ai/v1
- In production we'll be running the model locally in a vLLM (but this is outside the scope of this project)
- The App will collect information and fill a turso database (locally, it's like SQLite)
  * description
  * type of images: photo, screen-capture, images
  * tags: special element in the images
- THe app will have an interface for the user to be able to search images and filter the list or search resulty
- Image should be visible in the interface too.
